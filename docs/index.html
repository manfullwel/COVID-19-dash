Como Criar um Dashboard COVID-19

Um guia passo a passo para construir um dashboard interativo usando Python, Dash e Plotly

Visão Geral do Projeto

O que você vai aprender
  • ✓ Criar um dashboard interativo com Python
  • ✓ Manipular e visualizar dados com Pandas
  • ✓ Criar gráficos interativos com Plotly
  • ✓ Construir uma interface web com Dash
  • ✓ Fazer deploy da aplicação
Requisitos
  • ✓ Python 3.9+
  • ✓ Conhecimento básico de Python
  • ✓ Editor de código (VS Code recomendado)
  • ✓ Git instalado

2 Instalação das Dependências

Crie um arquivo requirements.txt com as seguintes dependências:

dash==2.14.1
dash-bootstrap-components==1.5.0
plotly==5.18.0
pandas==2.1.3
numpy==1.26.2
python-dotenv==1.0.0

Instale as dependências:

pip install -r requirements.txt

3 Criando o Dashboard

O dashboard é construído em módulos:

  • Configuração inicial e imports
  • Carregamento dos dados
  • Layout da interface
  • Callbacks para interatividade
