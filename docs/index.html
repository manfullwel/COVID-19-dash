plotly==5.18.0
pandas==2.1.3
numpy==1.26.2
python-dotenv==1.0.0

Instale as dependências:

pip install -r requirements.txt

3 Criando o Dashboard

O dashboard é construído em módulos para melhor organização e manutenção do código:

3.1. Configuração Inicial
import dash
from dash import html, dcc
import dash_bootstrap_components as dbc
import plotly.express as px
import pandas as pd

app = dash.Dash(__name__, external_stylesheets=[dbc.themes.BOOTSTRAP])
server = app.server
3.2. Carregamento dos Dados
# Carrega os dados do CSV
df = pd.read_csv('dados_covid.csv')
df['data'] = pd.to_datetime(df['data'])

# Prepara os dados para visualização
df_brasil = df.groupby('data').sum().reset_index()
3.3. Criando os Gráficos
# Gráfico de casos por dia
fig_casos = px.line(df_brasil, 
    x='data', 
    y='casosNovos',
    title='Novos Casos por Dia'
)

# Gráfico de óbitos por dia
fig_obitos = px.line(df_brasil,
    x='data',
    y='obitosNovos',
    title='Óbitos por Dia'
)
3.4. Layout do Dashboard

O layout é construído usando componentes do Dash Bootstrap:

app.layout = dbc.Container([
    dbc.Row([
        dbc.Col(html.H1("Dashboard COVID-19 Brasil"), 
                className="text-center my-4")
    ]),
    dbc.Row([
        dbc.Col(dcc.Graph(figure=fig_casos), md=6),
        dbc.Col(dcc.Graph(figure=fig_obitos), md=6)
    ])
], fluid=True)
3.5. Adicionando Interatividade

Os callbacks permitem atualizar os gráficos dinamicamente:

@app.callback(
    Output('grafico-casos', 'figure'),
    Input('select-estado', 'value')
)
def update_graph(estado):
    df_filtrado = df[df['estado'] == estado]
    fig = px.line(df_filtrado, x='data', y='casosNovos')
    return fig

4 Deploy

O deploy no Railway é simplificado usando o arquivo railway.toml:

# railway.toml
[build]
builder = "nixpacks"
buildCommand = "pip install -r requirements.txt"

[deploy]
startCommand = "python dashboard.py"
healthcheckPath = "/"
healthcheckTimeout = 100

[service]
port = 8080
Passos para Deploy:
  1. Crie uma conta no Railway
  2. Conecte seu repositório GitHub
  3. Configure as variáveis de ambiente:
    • PORT=8080
    • PYTHON_VERSION=3.9
  4. O deploy será automático a cada push no repositório
